### Added
- Project scaffold with React + TypeScript + Vite
- Force-directed graph visualization using D3.js
- Interactive nodes with click-to-inspect functionality
- Zoom and pan controls (mouse wheel + drag)
- Edge rendering with directional arrowheads
- Node icons by file type (component, util, hook, etc.)
- Zustand store for centralized state management
- Control panel for adjusting visualization settings
- Node inspector panel with file details
- AI integration via OpenRouter API
- File system walker with glob pattern support
- AST parser for TypeScript/JavaScript code analysis
- SQLite-based caching for parsed results
- Testing infrastructure with Jest and ts-jest
